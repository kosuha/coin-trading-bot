### ubuntu 명령어
- 백그라운드 실행: nohup python3 main.py > output.log &  
- 실행되고 있는지 확인: ps ax | grep .py
- 프로세스 종료(PID는 ps ax | grep .py를 했을때 확인 가능): kill -9 PID
